# piot
Raspberry Pi IoT Software - Slack Chatbot

## PiBot

PiBot is an Intranet Chatbot System. It is a software for Raspberry Pi written in Python. Main purpose is to provide services for office space intranet. 

Hardware:

- Raspberry Pi Zero W
- Monitor

Software:

- Raspbian 8
- Python 2
    - asciimatics
    - pyowm
    - slackclient
    - cPickle
